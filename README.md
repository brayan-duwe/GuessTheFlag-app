# Guess the Flag 🏳️

*This project was built using SwiftUI.*

This is a simple game where the player guesses the correct flag. The game shows the name of a country and the user must select the correct flag.

## Project files

- [ContentView.swift](/GuessTheFlag/ContentView.swift)
- [Assets](/GuessTheFlag/Assets.xcassets/)

The app structure is very simple, basically containing the ContentView and the Assets (the flags).
